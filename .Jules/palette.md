# Palette's Journal - Critical UX & Accessibility Learnings

## 2025-05-14 - Consistency in Input Feedback
**Learning:** For apps focusing on calm and mindfulness, visual consistency in helper elements like character counters is key. Using a consistent contrast level (e.g., `text-ink/70`) ensures accessibility while maintaining the soft aesthetic.
**Action:** Always check for similar input patterns (like Word Garden vs Mood Note) and unify their UX affordances (labels, counters, ARIA descriptions).
